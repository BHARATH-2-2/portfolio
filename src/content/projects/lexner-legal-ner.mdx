A comprehensive full-stack AI chatbot application for Legal Named Entity Recognition (NER). This system helps legal professionals extract and visualize structured information from legal documents.

## Core Features

- **AI-Powered Chatbot** — Conversational interface for legal document analysis
- **Legal NER** — Identifies and extracts key legal entities:
  - Judges
  - Counsels
  - Statutes
  - Cases
- **Graph Visualization** — Interactive vis-network graphs for exploring legal relationships
- **Session Management** — Context-aware conversations with session persistence
- **Structured Data Models** — Pydantic models for type-safe entity extraction

## Technical Architecture

### Backend (FastAPI)
- FastAPI framework for high-performance API endpoints
- Pydantic models for data validation and serialization
- Legal NER processing and entity extraction
- Session and context management

### Frontend (React + Tailwind CSS)
- Modern React application with component-based architecture
- Tailwind CSS for responsive, polished UI
- vis-network integration for interactive graph visualization
- Conversational chat interface

## Key Achievements

- **Structured Entity Recognition** — Accurate extraction of legal entities using Pydantic schemas
- **Interactive Visualization** — Dynamic graph exploration of legal case relationships
- **User Experience** — Intuitive conversational interface for legal document analysis
- **Full-Stack Integration** — Seamless communication between FastAPI backend and React frontend

This project demonstrates expertise in full-stack development, AI/ML integration, and creating specialized tools for legal technology applications.
